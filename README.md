## How to play

1. Go [here](https://maxadamski.com/galactica), or download the repo, and open `index.html`.
2. Enter a server address and port
3. Enjoy :)

## How to run a server

These commands compile and run the server on port 6060:

```sh
cd server
make build
./main 0.0.0.0 6060
```
